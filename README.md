# KommandDesk

> Turn scattered thoughts into executable plans.

## What is KommandDesk?

KommandDesk is a full-stack AI workflow automation platform that converts 
unstructured notes and goals into structured, reviewable tasks. It solves 
the problem of disorganized, unacted-on thoughts by giving users a system 
to capture, extract, approve, and track their work.

## Core Workflow

messy note → structured task suggestions → review/approve/reject → save → dashboard

## Tech Stack

- Frontend: Next.js, React, TypeScript, Tailwind CSS
- Backend: Java, Spring Boot, REST APIs
- Database: PostgreSQL
- Validation: Zod, Java DTO validation
- AI: Claude API or OpenAI API (coming later)
