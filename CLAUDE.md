# CLAUDE.md — KommandDesk

## What is this project?

KommandDesk is a full-stack AI workflow automation platform that converts 
unstructured notes into structured, reviewable tasks.

## Tech Stack

- Frontend: Next.js, React, TypeScript, Tailwind CSS
- Backend: Java, Spring Boot, REST APIs
- Database: PostgreSQL
- Validation: Zod (frontend), Java DTO validation (backend)

## Project Rules

- Do not build the whole project at once
- Work one milestone at a time
- Use mock data before real AI
- Never expose API keys or personal data in demo mode
- All AI-generated tasks must be approved before saving to the database

## Core Workflow

messy note → task suggestions → review/approve/reject → save → dashboard

## Two Modes

- Private Mode: real user workflow and personal tasks
- Demo Mode: safe sample data for portfolio visitors

## Commit Style

- chore: setup or config changes
- feat: new feature
- fix: bug fix
- docs: documentation only
- refactor: restructuring code
- test: adding tests
